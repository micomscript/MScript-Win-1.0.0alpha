■MicomScript概要

MicomScriptはマイコン制御用スクリプト言語です。
C言語に似た制御構文を持つスクリプト言語に仕上げました。


●ファイル内容
	・bin				バイナリコードフォルダ
	・doc				ドキュメントフォルダ
	・samples			サンプルスクリプト
	・src				Windows版ソースコードフォルダ
	・LICENSE.txt		LICEMSEファイル
	・README.txt		READMEファイル
